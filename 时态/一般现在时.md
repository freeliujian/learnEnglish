主系表结构一般现在时

1. 人称代词主格 + be + 形容词
2. 名词 + be + 形容词
3. 人称代词主格 + be + 名词
4. 名词 + be + 名词

### 人名词代词主格
```
I am happy.
you ary beautiful.
it is cheap.
```
### 名词做主语
```
Joe is busy
Changsha is big
Bejing and Changsha are big.
```
### 人称代词 + be + 名词
```
I am a doctor.
It is my room.
```

### 名词 + be + 名词
```
joe is my brother.
```

## 一般疑问句
把一般现在时的be动词放在句首.
```
you are a teacher.
Are you a teacher?
```
